# Privacy Policy for Discover Minds Extension

## Data Collection and Usage
This extension does not collect or store any personal information. We use the following permissions for specific purposes:

### Permissions Justification
- **Host Permission**: Required to detect installation events on web pages
- **Notifications**: Used to display installation status notifications to users
- **Storage**: Used to store user preferences and extension settings locally
- **Remote Code**: The extension executes only packaged code; no external code is loaded

## Single Purpose
The Discover Minds extension serves a single purpose: to notify users about successful installations and provide installation status updates.
