# Privacy Practices

## Required Permission Justifications

### Host Permissions (`https://*.discoverminds.ai/*`)
Our extension needs to communicate with our secure API endpoints to:
- Authenticate users
- Search and retrieve professional profiles
- Process AI-powered matching algorithms
- Enable secure user connections
Without this permission, core functionality would not work.

### Storage Permission
This permission is essential for:
- Storing user authentication tokens securely
- Saving user preferences and settings
- Maintaining search history for better recommendations
- Caching frequently accessed data to improve performance
No sensitive data is stored unencrypted.

### Notifications Permission
We require notifications to:
- Alert users about new relevant professional matches
- Notify when someone wants to connect
- Send important security alerts
- Provide updates about connection requests
Users can disable notifications at any time.

### Remote Code Usage
Our extension:
- Contains all code within the package
- Does not load or execute external code
- Updates only through the Chrome Web Store
- Uses only local scripts for security

## Single Purpose Description
Discover Minds is a focused professional networking tool that serves one clear purpose: enabling users to find and connect with relevant professionals based on AI-powered expertise matching. All permissions and features directly support this core functionality of professional discovery and networking.

## Data Usage & Collection Policy
- We only collect data necessary for the extension's core functionality
- All data collection and processing complies with Chrome Web Store policies
- User data is encrypted during transmission and storage
- No personal data is shared with third parties
- Users can request complete data deletion
- We maintain transparency about all data practices

I certify that this extension complies with all Chrome Web Store Developer Program Policies.
